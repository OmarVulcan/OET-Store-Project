OET Store - MySQL Database Configuration

‚ö†Ô∏è Important: Make sure XAMPP or WAMP is installed and both Apache and MySQL services are running before proceeding.

Step 1: Prerequisites

Ensure you have the following installed and running:

  • XAMPP or WAMP Server
  • Apache Web Server (running)
  • MySQL Database Server (running)
  • PHP 7.4 or higher

Step 2: Database Configuration

The database will be created with the following settings:

Database Name: oet_store

Host: localhost

Username: root

Password: (empty)

If your MySQL uses different credentials, update them in includes/db.php

Step 3: Tables to be Created

The following tables will be created:

  • users - User accounts with secure password hashing
  • lists - User watchlists
  • list_items - Items in watchlists
  • cart - Shopping cart items
  • orders - Completed orders
  • order_items - Items in orders
